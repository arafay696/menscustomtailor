# Men's Custom Tailor (MCT)

Create custom shirts at menscustomtailor.com! Design your own shirts.

## Features!

- Select Your Fabric
  - Fabric by Category
  - By Color
  - By Price
- Design Your Shirt
  - Sizing
  - Choose Shirt Type
  - Choose Collar/Cuff
  - And more..
- PayPal integration
- Cart Management 
- And more... have a try!

## Tools

- Laravel 5.1
- Gulp
- JQuery
- CSS
- HTML
